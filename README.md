# CyberSafe
Store your home sensors data on your own mini-storage RPi cloud and view real time data using PyCharts from anywhere, anytime

<b>Hardware Used:</b>
<ul>
  <li>Raspberry Pi</li>
  <li>Groove Kit</li>
  <li>Intel Edison Board</li>
</ul>

<b>Software Used:</b>
<ul>
  <li>Python</li>
  <li>Flask</li>
  <li>PyCharts</li>
  <li>Putty - for SSH (No need on MacOS)</li>
</ul>
